---
title: "缺少文档？"
description: "告诉我们你希望补充的 iPolloWork 文档。"
---

我们会持续补齐 iPolloWork 文档。若某个功能尚未说明清楚，请通过以下方式告诉我们：

- [发送邮件给团队](mailto:team@ipolloworklabs.com?subject=missing%20documentation&body=hey%20my%20github%20is%20%7Busername%7D%2C%20I%27m%20interested%20in%20%7Bfeature%7D%20from%20ipollowork%20but%20I%27m%20missing%20documentation%20on%20it%2C%20could%20you%20create%20it%3F%0A%0AThanks)，说明你想使用的功能。
- 在 [GitHub Issues](https://github.com/Devin-AXIS/iPolloWork/issues) 中提出请求。

文档源文件位于 [iPolloWork 仓库的 `packages/docs`](https://github.com/Devin-AXIS/iPolloWork/tree/main/packages/docs)。如果愿意协作，也欢迎提交新的 `.mdx` 页面或改进现有内容的 Pull Request。
