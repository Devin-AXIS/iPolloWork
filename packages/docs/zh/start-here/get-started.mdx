iPolloWork 是一个不止用于编程的开源可用智能体工作空间。你可以用 AI 创建代码、文档、网站、演示文稿、设计和视频，并直接编辑结果，而不是每次都重新发起提示。大多数用户可以从 [iPolloWork 应用](https://ipolloworklabs.com/download)开始：连接常用模型，再按需添加 Skills、插件或 MCP 服务器。

## 开始

连接你的 LLM 提供方并发送第一条消息：

<CardGroup cols={3}>
  <Card title="连接模型提供方" icon="plug" href="/zh/start-here/connect-your-stack/sign-in-with-chatgpt">
    登录 ChatGPT，或在几步内接入自己的 LLM 密钥。
  </Card>
  <Card title="添加 MCP" icon="puzzle-piece" href="/zh/start-here/connect-your-stack/add-an-mcp-server">
    几分钟内把自定义 MCP 服务器接入工作空间。
  </Card>
  <Card title="控制浏览器" icon="globe" href="/zh/start-here/do-work-with-it/control-the-browser">
    让 iPolloWork 在 Chrome 中为你执行网页任务并截取页面。
  </Card>
</CardGroup>
