---
title: "控制浏览器"
description: "如何使用 iPolloWork 自动化内置浏览器。"
---

iPolloWork 目前通过内置的 iPolloWork Browser 实现计算机使用能力。这并不代表它已经可以完整控制 Ubuntu、macOS 或 Windows 的桌面应用。

iPolloWork 可以在这个浏览器中打开网页、点击、填写表单、读取页面内容并截图。若你连接的是远程 iPolloWork 服务器，浏览器会运行在对应的远程 Worker 上。

目前支持的方式是使用第一方 `iPolloWork Browser` 扩展。

## 要求

- 使用桌面应用，或使用提供 iPolloWork Browser 的远程 Worker。
- 在 `Settings > Extensions` 中保持 `iPolloWork Browser` 启用。

完成后回到会话页面，直接让 iPolloWork 打开一个网站即可。
