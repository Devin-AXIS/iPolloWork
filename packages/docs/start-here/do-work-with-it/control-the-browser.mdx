---
title: "Control the browser"
description: "How to use iPolloWalk to automate the built-in browser"
---

Computer-use in iPolloWalk currently works through the built-in iPolloWalk Browser. It does not yet mean full desktop control of Ubuntu, macOS, or Windows apps.

iPolloWalk can open pages, click, fill forms, read page content, and take screenshots inside that browser. If you connect to a remote iPolloWalk server, the browser runs on that remote worker.

Today, the supported path is the first-party `iPolloWalk Browser` extension.

## Requirements

- Use the desktop app or a remote worker that exposes the iPolloWalk Browser.
- Keep `iPolloWalk Browser` enabled in `Settings > Extensions`.

Then go back to the session view and ask iPolloWalk to open a website.
