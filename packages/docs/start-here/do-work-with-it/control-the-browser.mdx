---
title: "Control the browser"
description: "How to use iPolloWork to automate the built-in browser"
---

Computer-use in iPolloWork currently works through the built-in iPolloWork Browser. It does not yet mean full desktop control of Ubuntu, macOS, or Windows apps.

iPolloWork can open pages, click, fill forms, read page content, and take screenshots inside that browser. If you connect to a remote iPolloWork server, the browser runs on that remote worker.

Today, the supported path is the first-party `iPolloWork Browser` extension.

## Requirements

- Use the desktop app or a remote worker that exposes the iPolloWork Browser.
- Keep `iPolloWork Browser` enabled in `Settings > Extensions`.

Then go back to the session view and ask iPolloWork to open a website.
