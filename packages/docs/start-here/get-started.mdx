iPolloWork is a source-available agent workspace built for more than coding. Create code, documents, websites, presentations, design, and video with AI, then directly edit the result instead of starting another prompt. Most people should start with the [iPolloWork app](https://ipolloworklabs.com/download), connect their preferred model, and add Skills, plugins, or MCP servers as needed.
