# iPolloWork installer

Install and start the latest official iPolloWork desktop app with:

```bash
npx -y ipollowork
```

The installer detects macOS, Windows, or Linux and downloads the matching official asset from [iPolloWork Releases](https://github.com/Devin-AXIS/iPolloWork/releases).

Use a specific desktop release when needed:

```bash
npx -y ipollowork --release v0.17.26
```

Node.js 20 or newer is required to run the installer.
