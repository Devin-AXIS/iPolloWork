# calendar-meet-link — Google Calendar events can include Meet links

1. I ask iPolloWork to schedule a Google Calendar meeting and include a Google Meet link. iPolloWork uses the connected Google Workspace account instead of sending me to Calendar manually.

2. The event is created with the right title, attendee, and time, and iPolloWork confirms both the calendar event link and the Google Meet link.

3. If an event already exists without conferencing, I ask iPolloWork to add a Meet link. iPolloWork updates the existing event instead of creating a duplicate.

4. iPolloWork confirms the updated event now has a Google Meet link, so the invite is ready to send or join without any manual Calendar edits.
