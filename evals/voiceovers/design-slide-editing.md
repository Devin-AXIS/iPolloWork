# design-slide-editing — edit a multi-page deck without losing place

1. I move to the second slide first, then turn on Edit page. The same second slide remains visible, and the compact deck control confirms where I am.

2. While editing, the deck's own next button still advances to slide three. I return to slide two, type directly into its heading, and drag that heading without leaving the paged editor.

3. The same Design toolbar now offers the clean scrolling HTML file as well as the slide-by-slide PDF and PowerPoint exports.
