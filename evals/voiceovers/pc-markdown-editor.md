# pc-markdown-editor — Markdown files edit naturally in the PC client

1. I open a Markdown file in the existing right-side browser and it reads like a clean document while remaining directly editable.

2. I type a slash and a compact menu appears at my cursor, where I can filter and insert Markdown blocks without leaving the keyboard.

3. I select text and a small floating toolbar appears with the Markdown-safe basics: bold, italic, strikethrough, inline code, link, and block type.

4. I format the text, pause, and see that the file saves automatically; after switching away and reopening it, the same Markdown content is still there.

5. Images, fenced code, and GFM tables stay lightweight Markdown underneath while rendering directly in the editor, so the file reads like a document instead of source text.

6. I hover a rendered image and choose Replace to update its URL or description without hunting through Markdown syntax.
