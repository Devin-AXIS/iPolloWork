# creative-studio-ipwt — import and edit a self-contained Creative template

1. I start a new Design task, open Website templates, and import `creative-studio.ipwt`; iPolloWork identifies it as a local MIT template named Creative Studio.

2. I choose Creative Studio and complete its short Design brief, selecting a new primary color before opening the editable page.

3. The page renders navigation, hero, services, six portfolio projects, a call to action, contact details, and a footer using only files packaged in the template.

4. I turn on Edit page, select the hero heading, change its text, and open the Design System to change the global theme while the portfolio images remain editable local assets.

5. I save the page, close Design, and reopen it; the edited heading and theme remain in the workspace-specific Design session.
